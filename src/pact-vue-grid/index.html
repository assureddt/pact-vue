Vue Grid Example

This demo page should function if you run with: `npm run dev`

Note: the sample is not complete, as it lacks a proper back-end, but it should:

  • Be bootstrap 5 styled and have font-awesome icons
  • Contain a few row of sample data pulled from a local json file
  • Have interactable buttons that trigger edit/delete-ish behaviour when clicked
  • The edit page should embed a markdown editor for the Description field, as a demonstration of how to use that feature
  • Not react to column re-ordering
  • Not perform those edit and delete options
